# RecipeBird 菜谱机

让咕咕机打印指定的下厨房的菜谱。感谢 Windfarer 的咕咕机 Python API (https://github.com/Windfarer/pymobird)

已经部署到 Heroku 上，访问 <https://recipebird.herokuapp.com/> 即可直接使用。
